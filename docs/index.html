Hindsight

DevTools you didn't open.

A Chrome extension that passively records every network request, console error, action and screenshot in the background — so when something breaks, you can scrub back through time and ship a faithful bug report without re-running anything.

Install for Chrome View on GitHub

Three brand promises

Every architectural decision in Hindsight defends one of these. They're load-bearing — not aspirational marketing copy.

No information loss. Hindsight never silently truncates, denoises, or "smart summarizes" your capture. What got captured is what you can share.
Your data does not leave your machine unless you choose. Zero outbound network requests except ones you explicitly initiate — a webhook POST, a web-intent navigation, a file download. No telemetry. Ever.
Zero setup, useful from minute one. Defaults work the moment you install. Settings exist for power users. The killer feature is a single-file HTML replay bundle a teammate opens in any browser, no recipient setup.

What it captures

Default-on the moment you install. Tier 4 captures (cursor / scroll / periodic screenshots) light up only when you explicitly hit ● Record.

Network

fetch, XMLHttpRequest, WebSocket frames, SSE. Full request/response with headers and bodies. Sensitive headers masked at capture time.

Console

error, warn, info, plus unhandled rejections and uncaught errors with stack traces.

User actions

Clicks, form input changes (passwords always masked), SPA route changes. Tag, accessible name, position — not coordinates.

Screenshots on error

Single JPEG frame at the moment of the failed request. Throttled to one shot per tab per two seconds.

Performance signals

Long tasks > 100 ms, cumulative layout shift, white-screen heuristic — the moments your app feels janky show up on the timeline.

Recording mode

● Record kicks in cursor trail, throttled scroll, and 2-second screenshots — for the "show me what they did" reproductions.

Capture-time masking, not export-time

The original value of a sensitive field is never written to your local storage — replaced with ***MASKED*** before persistence. So even if your machine is compromised, the bearer tokens aren't there to leak. 

{
  "Accept": "application/json",
  "Authorization": "***MASKED***",
  "Cookie": "***MASKED***",
  "Content-Type": "application/json"
}

Default rules cover Authorization, Cookie, Set-Cookie, X-API-Key, X-Auth-Token, Proxy-Authorization, credit-card numbers (Luhn-validated), and Turkish national IDs. Add your own regex in Settings → Privacy. Disable an individual rule when you genuinely want the raw value in your own captures.

The replay bundle

Click "Save as replay bundle" and Hindsight writes a single hindsight-<host>-<timestamp>.html file. Drag it into any browser tab and the recipient sees the full session — scrubber, narrative, request/response panels — with no extension, no service, no login. Works offline.

How it actually works

  1. Install & forget. The toolbar icon shows a quiet event counter while you browse. No popups, no prompts.
  2. Something breaks. Click the icon. The side panel opens with a scrubber across the last N events, cluster banners for repeated failures, and detail panes for any request you tap.
  3. Pick a slice. Use the dual-handle scrubber to clip a time window. Filter by Failed / API / All. Search by URL, type, or response body. Pin a single host.
  4. Share it. Copy bug report (markdown + screenshot to clipboard), → Slack/Discord/Teams webhook, → GitHub Issue, ⤓ Download replay bundle, ⤓ ZIP everything. Privacy preview modal shows what's about to leave the machine before it does.

Install

Coming to the Chrome Web Store. While we're under review, you can load the latest packaged build directly:

  1. Download the release. Grab hindsight-v0.5.0.zip from the GitHub releases page. Unzip into a folder you'll keep around.
  2. Open chrome://extensions. Toggle Developer mode on in the top-right corner.
  3. Click "Load unpacked". Pick the unzipped folder. Pin the Hindsight icon to your toolbar.
  4. Browse normally. Hindsight captures in the background. When something breaks, click the icon — the side panel is your inspector.
